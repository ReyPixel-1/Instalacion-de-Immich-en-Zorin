📸 Instalador de Immich para Zorin OS, Debian y derivados
Script automatizado para instalar tu propia nube de fotos personal. Optimizado para Zorin OS, Ubuntu y derivados de Debian.

🚀 Instalación automática
Copia y pega este comando en tu terminal para configurar todo:

```bash
mkdir -p ~/scripts && wget -qO ~/scripts/instalar_immich.sh https://raw.githubusercontent.com/ReyPixel-1/Instalacion-de-Immich-en-Zorin/main/instalar_immich.sh && chmod +x ~/scripts/instalar_immich.sh && ~/scripts/instalar_immich.sh```

🛠️ Qué instala este script
Docker Engine y Compose: Repositorios oficiales.

Immich: Configuración completa en ~/immich.

Lanzador de Menú: Crea un acceso directo en tu menú de aplicaciones.

📁 Estructura de archivos
Script: /home/$USER/scripts/instalar_immich.sh

Lanzador: /home/$USER/.local/share/applications/mis apps/instalar-immich.desktop
